This collection is testing basic CLI behavior without involving any subcommands.

Also includes a bunch of tests performed once on an arbitrary sub-command, for situation when the tested behavior is shared by all subcommands. The arbitrary sub- command is managers, as it is a safe read-only operation supposed to work on all platforms, whatever the environment.
